# apollo

An open-source island for your Macbook notch

Features:
Clipboard Manager, "Clip".
Write quick notes, "Jot".
Hold any file to move/share it, "Box".
Start a stopwatch or timer with HUD, "Chrono".


Installation:
Go to the "Releases" page to download the latest release and select the .dmg file.
Once downloaded, open it to add the installer to your desktop. Install by dragging "Apollo" to the applications folder in the menu. 
You'll need to attempt to open it, which it will then say it can't be opened. Head to your settings->privacy & security and scroll all the way to the bottom.
It should say "Open Anyway" for Apollo. Click that. After authenticating, it should open right away.

Removing Apollo:
Open your "Applications" folder.
Right click on the "Apollo" app.
Select "Move to Trash".
Gone.
